contoh Kasus : 

1. Kita mengkonversi tipe data int ke float64 dengan menggunakan float64(x).
2. Kita mengkonversi tipe data float64 ke int dengan menggunakan int(a).
3. Kita mengkonversi tipe data string ke int dengan menggunakan strconv.Atoi(s). Perhatikan bahwa Atoi mengembalikan dua nilai, yaitu nilai konversi dan error jika terjadi kesalahan.
4. Kita mengkonversi tipe data int ke string dengan menggunakan strconv.Itoa(d).
